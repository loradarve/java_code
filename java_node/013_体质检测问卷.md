## 计分与体质判定说明
### 1.计分规则
依据九种体质分项归集对应题目得分，**单项体质总分越高，该体质偏颇倾向越明显**，得分≥40提示明显偏颇。
- **平和质**：反向计分，总分越高体质越均衡健康；
- **气虚、阳虚、阴虚、痰湿、湿热、血瘀、气郁、特禀**：正向计分，分值越高对应偏颇越重。

### 2.九大体质对应归类（国标原版）
1. **平和质**：1题（精力充沛），分值越高体质越健康
2. **气虚质**：2、3、4题（易累、气短、心慌）
3. **气郁质**：5、6、7题（低落、紧张、敏感脆弱）
4. **痰湿质**：8、23、24题（身体困重、口黏、腹松软）
5. **阳虚质**：9、10、12题（腰背怕冷、畏寒、不耐冷）
6. **阴虚质**：11、17、19题（潮热、皮肤干、面颊潮红）
7. **特禀质**：13、14、15、16题（易过敏、荨麻疹、抓痕、频发喷嚏）
8. **血瘀质**：18、21、22题（身体隐痛、面色暗沉、唇暗）
9. **湿热质**：20、25、26题（面部出油、小便灼热、下体潮湿）
