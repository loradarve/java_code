# PHQ-15 躯体病症评估健康量表（适配优化：作答周期近1个月）
## 一、量表简介
PHQ-15是国际通用躯体化症状筛查量表，共15项躯体相关问题，多用于筛查躯体不适、身心共病相关症状，广泛应用于临床体检、心理科健康测评。
### 作答说明
依据**近一个月**自身实际症状频次打分：
0=没有、1=有一点、2=有一些、3=非常多

|序号|测评题目|0(没有)|1(有一点)|2(有一些)|3(非常多)|
|----|----|----|----|----|----|
|1|心悸或心跳加快|0|1|2|3|
|2|呼吸困难|0|1|2|3|
|3|胸闷或胸部疼痛感|0|1|2|3|
|4|头晕、头昏或突发晕倒感|0|1|2|3|
|5|身体虚弱、浑身乏力|0|1|2|3|
|6|精神紧张、容易焦躁激动|0|1|2|3|
|7|手脚发麻或肢体刺痛感|0|1|2|3|
|8|周身肌肉酸痛、躯体疼痛|0|1|2|3|
|9|胃部不适、腹胀或消化不良|0|1|2|3|
|10|便秘、排便不畅困难|0|1|2|3|
|11|口腔、咽喉干燥缺水|0|1|2|3|
|12|尿频、突发尿急表现|0|1|2|3|

|14|经期紊乱、经期腹痛（女性作答）|0|1|2|3|
|15|其他不明原因躯体不适|0|1|2|3|

## 二、计分与结果判定（总分范围：0～45分）
|总分区间|躯体症状分级|参考建议|
| ---- | ---- | ---- |
|0～4分|无/轻度躯体不适|日常作息调养即可，规律饮食作息|
|5～9分|轻度躯体症状|优先调整作息，观察1～2个月症状变化|
|10～14分|中度躯体症状|建议全科/中医科查体，排查器质性问题|
|15～19分|中重度躯体症状|前往医院内科或临床心理科就诊检查|
|20～45分|重度躯体症状|尽快完善全身体检+心理专科评估|

> ⚠️小程序免责备注：本量表仅作躯体健康筛查参考，**不能替代临床检查与医师诊断**，躯体不适持续反复请前往正规医院就诊。
