## 一、DIFY的介绍、准备与安装
1. Dify 是一个开源的「低代码大模型应用开发平台」，核心作用是让普通人 / 小团队也能快速做出可用的 AI 应用，不用从零写复杂后端与 AI 逻辑。应用领域：智能客服、企业知识库、内容工厂、自动化助手。
2. 安装Docker：搜索docker官网，选择windows版本安装即可
    > x64 CISC(复杂指令集) 单核性能强大，我们的电脑基本都是这种，换言之，无视.arm，选amd的即可
    > ARM格式 RISC(精简指令集) 低功耗多核，多用于轻量级系统如手机平板
3. 准备WSL
   > WSL（Windows Subsystem for Linux） 是 Windows 10/11 自带的官方功能，在windows系统下建立Linux子系统，完全直连计算机硬件，正在逐渐替代虚拟机成为Linux学习使用的主流方式
   * wim+R>>control>>控制面板中选择程序>>启用或关闭windows功能>>下滑最下面勾选linux子系统重启电脑
   * 在Microsoft store中搜索下载安装Ubuntu和Windows Terminal，然后Ubuntu输入账号密码完成，终端设置默认打开乌邦图，这俩都安装在c盘某处，打开从Microsoft Store搜索打开
4. 设置镜像(无需登录，直接跳过)

## 二、安装ollama
<details>
<summary>点击查看多个命令行窗口的区别</summary>

1. 命令提示符 CMD
最老的 Windows 原生黑窗口，几十年历史。
* 特点：语法老、功能弱、只支持传统 DOS 命令
* 能干：ping、ipconfig、删文件、简单文件夹操作
* 不能干：不支持 PowerShell 脚本、不支持 irm 在线安装
你记住：老旧工具，现在基本不用特意开它
2. PowerShell
微软新版终端，CMD 的超级加强版
* 完全兼容 CMD 所有命令
* 支持高级脚本、系统运维、在线安装软件（Ollama、各种工具）
* 支持面向对象、功能强很多
你记住：
装 Ollama、Docker、开发运维、执行网上复制的 Windows 脚本 → 必须用 PowerShell
3. 终端（Windows 终端 / Windows Terminal）
它不是新的命令内核，只是一个高颜值壳子
里面可以同时装：
* CMD
* PowerShell
* WSL Ubuntu
* Anaconda
随时切换标签页，界面好看、支持配色、分屏
理解：
Windows 终端 = 一个集成了各种命令行的豪华外壳
本身没独立语法，只是把上面几个装在一起。
4. Anaconda Prompt
专门给 Python /conda 量身定做的专属窗口
本质是 改了环境变量的 CMD
一打开就自动加载 conda、python 环境
不用自己配置 python 路径
只能专心干：创建虚拟环境、装 Python 库、跑 Python 项目

### 温度=2 问题：写一段关于哲学：存在主义的介绍，写的有趣一些
1. 模型：llama3.2:3b
> existentialism是一种哲学派别，强调个体在面对现实世界时的自由和责任。这个词本身就很有趣了，来自拉丁语“existeri”，意思是“存在”。
>你可能听说过萨特的名言：“人是自由的，人注定自由，人背负着自由的命运。” 这个概念就代表了存在主义的精髓：人类不被天生的生物学、社会环境或宗教等因素决定，而是通过自己的选择和行动来构建自己的意义。
>生活在这样一个世界中，似乎没有绝对的道德标准，甚至没有绝对的真理。你需要自己去创造你的价值观，定義你自己。这种“自我创造”让人有着一种自由感，但同时也给了人一种深深的责任感。
>存在主义并不仅关注人类的存在，还关注人与现实世界之间的关系。这意味着你需要面对自己的恐惧、疼痛和不确定性，而不是逃避或否定它们。萨特说：“不在苦难中获得理解，很奇怪，因为任何真正的理解都应该基于苦难的经历。” 在这种背景下，存在主义者会寻求真实的理解，并对生活采取现实和深刻的态度。
>然而，存在主义并不是一种“人生悲剧”的哲学。相反，它是一种积极、自我激励的方法，鼓励人们去创造自己的意义，并在面对现实世界时勇敢地前行。
2. 模型：通义千问qwen3-max
>想象一下，你一睁眼，发现自己被“空投”到这个世界上——没有使用说明书，没有预设角色，也没有客服热线。没人告诉你该成为谁、该信什么、该往哪走。更糟的是，世界看起来既冷漠又荒诞：上班打卡像推石头上山的西西弗斯，刷手机三小时却感觉啥也没干，连点个外卖都要纠结半小时……
>欢迎来到存在主义的世界——哲学界最“人间真实”的流派。
>存在主义不跟你聊宇宙真理或神是否存在，它直勾勾盯着你问：“嘿，既然你已经在这儿了，打算怎么活？”
>它的核心观点简单粗暴：存在先于本质。意思是，人不是生来就带着“人生剧本”的。不像一把刀注定用来切菜，你是先“存在”了，然后才通过一次次选择——是躺平还是奋斗？说真话还是随大流？爱一个人还是伤害TA？——亲手把自己“捏”成某种样子。
>这听起来很自由，对吧？但自由的代价是责任。你不能再怪原生家庭、社会压力或星座运势了。你的懦弱、逃避、甚至“摆烂”，都是你选的。萨特说得更狠：“人被判自由。”——自由不是礼物，是判决！
>而加缪则盯着世界的荒诞冷笑：生活本无意义，就像西西弗斯永远推着滚下山的石头。但他说：“我们必须想象西西弗斯是幸福的。”因为明知无意义，仍选择坚持，本身就是一种反抗和胜利。
>所以，存在主义不是教你emo，而是让你在混乱中挺直腰板：既然世界不给你意义，那就自己造一个！用行动，用选择，用真实地活着。
>简言之：
>没人安排你的人生——
>但正因如此，
>你才是自己故事里唯一的主角。
3. 模型：deepseek-v4-flash
> 好的，没问题。我们来聊聊存在主义——这门听起来就很高冷，其实无比接地气的哲学。
>想象一下：你刚参加完一个全是陌生人的派对，累得要死，一个人走在深夜的回家路上。霓虹灯闪得让人心烦，你突然冒出个很欠揍的念头：“我这是在干嘛？这一切到底有什么意义？”
>恭喜你，就在那一瞬间，你已经一脚踏进了存在主义的大门。
>为什么这么说？因为存在主义不聊“世界的本质是物质还是精神”这种硬核问题。它只关心一件事：一个活生生、有血有肉的人，在面对这个毫无意义、冰冷荒诞的世界时，应该怎么活？
>第一幕：人被“扔”进这个世界
>存在主义的第一句“咒语”是萨特说的：“存在先于本质。”
>翻译成人话就是：你，作为一个活生生的人，是先被莫名其妙地“扔”到这个世界上的。在你出生之前，没有任何一本“人生说明书”告诉你应该成为什么样的人。你不是一块出厂就标好了“用途”的螺丝钉。
>不像一把椅子。椅子在造出来之前，工匠脑子里已经有了“椅子”的蓝图——它的“本质”（用来坐）先于它的“存在”（被造出来）。但人不是！我们是先“存在”了，然后所有的一切，比如你是善良还是冷漠，是勇敢还是懦弱，是996的社畜还是浪迹天涯的背包客，都由你自己说了算。
>没有上帝，没有命运，没有“天性使然”。你每一次的选择，都在定义你是什么样的人。这听起来是不是很爽？但别急，爽的背后藏着巨大的责任。
>第二幕：自由的“诅咒”
>因为“存在先于本质”，所以你获得了绝对的自由。你可以选择成为任何一种人，没有借口，无法推卸。
>这正是存在主义最残酷也最迷人的地方。你会感到一种深深的焦虑，萨特叫它“自由的重负”。
>想象一下，你站在悬崖边，你知道自己可以跳下去，这是你的自由。但你也知道，你完全可以选择不跳。这种“一切皆有可能”的眩晕感，就是自由的重负。
>这种焦虑的体验，加缪（另一位存在主义大佬）用更有画面感的概念来形容：荒诞。
>加缪说，我们就像那个日复一日推石头上山的西西弗斯。每天早上醒来，挤地铁，上班，开会，加班，下班，刷手机，睡觉。第二天再来一次。这种“重复”和“无意义”的碰撞感，就是荒诞。
>世界本身没有意义，但我们偏偏是个追求意义的“意义制造机”。 这就是人与世界之间最根本的矛盾。世界问：“你要什么意义？” 我们咆哮：“老子就要个意义！” 然后世界面无表情地回你：“没有。”
>这咋整？躺平？自杀？怨天尤人？
>最终幕：活出个“人样”来
>存在主义哲学家们的回答是：不需要世界给你意义，而是由你，去给世界一个意义。
>萨特说：“人是自由的，人注定自由，人背负着自由的命运。” 因为自由，所以你必须为自己的每一个选择负责。你的每一次选择，不仅仅是在塑造你自己，也是在为全人类树立一个榜样。当你说“我选择诚实”时，你其实是在说“我主张，在这个世界上，人应该诚实。”
>这就是“承诺”和“责任”的力量。
>而加缪在《西西弗神话》里给出了更燃的回答。他说，我们应该想象西西弗斯是幸福的。为什么？因为当他明知那块石头会滚下来，却依然走下山谷，重新俯身，用尽全身力气再次推起它时，他那蔑视荒诞命运的意志，就已经战胜了荒诞本身。他通过“重复而坚定的行动”，赋予了那看似无意义的劳作以意义。
>所以，我们该怎么用存在主义“拯救”生活？
>别再问“人生的意义是什么”，而是问“我今天的选择，让我的生命有了什么意义？” 你的每一次选择，每一次“作为”，都是在回答这个问题。
>别想当“圣母”或“圣父”。你可以自私，可以软弱，可以犯错。但前提是你得坦然接受：这就是我选的，我认，并且我愿意为此承担后果。 ——这本身就是一种“英雄主义”。
>当你感到焦虑时，恭喜你。这证明你清醒地意识到了自己的自由和责任，意识到自己不是在随波逐流。焦虑是自由的眩晕，是觉醒的标志。
>在荒诞的世界里，做一个“推石头的西西弗斯”。上班、学习、做家务……这些看似重复的事情，当你带着“这是我的选择，我要做好它”的姿态去做时，它们就不再是荒诞的折磨，而是你反抗荒诞的武器。
>最后总结一下：
>存在主义不是教你悲观厌世的哲学，恰恰相反，它是一碗最“燃”的鸡汤——让你在没有上帝、没有剧本、没有意义的世界里，真诚地、勇敢地、充满激情地去生活，去创造，去热爱。 它告诉你：你的人生，你就是绝对的主宰，也是绝对的负责人。
>现在，你还觉得那个深夜街头感到空虚的自己很丧吗？不，也许那一刻，你正离真正的“存在”最近。

        * 链接工具database，输入`sql`，输出`json`
        * 输出变量：由`json`组成的`array`
    - 代码执行节点：**array转化为列表**  
        * 输出循环体结果字典
    - 输出节点：**提示：SQL计算完毕**
    - LLM节点：**汇总结果**
        * 输入变量：循环体输出 + 上一个LLM结果 + 用户问题
        * 提示词`system`
        * 输出变量：`text`
    - 代码执行节点：**图标可视化*
        * 调用工具`ECHART` 
    - 输出节点：**提示：图表生成完毕**
    - 条件分支节点：**判断生成何种图表**
        * IF 图参数=0 --> 工具：无  -->提示：无
        * ELIF 线性图 --> 工具：线性图 --> 提示：线性图
        * ELSE 柱状图 --> 提示：柱状图 --> 工具：柱状图
        * ELSE 饼图 --> 提示：饼图 --> 工具：饼图
        * 其中工具的参数有：标题、数据、坐标
