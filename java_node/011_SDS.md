## 计分与结果参考
1. **粗分计算**：20题得分相加得到原始粗分，**标准分=粗分×1.25后取整数**
2. **结果划分**
- 标准分＜53分：无抑郁相关倾向，保持作息调养即可
- 53～62分：轻度抑郁倾向，建议自我调节，必要时咨询心理从业者
- 63～72分：中度抑郁倾向，优先前往心理门诊专业评估
- ＞72分：重度抑郁倾向，尽快到正规医院精神/心理科就诊

> 免责备注：本量表仅为心理健康筛查参考，**不能作为临床诊断凭据**，症状持续不适请就医。
