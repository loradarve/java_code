## 二、测验说明
### 1. 测验目的
本测验从感觉、情感、思维、意识、行为、生活习惯、人际关系、饮食睡眠等多维度，评定个体是否存在心理症状及严重程度，可筛查潜在心理困扰人群；**不适用于躁狂症、精神分裂症确诊筛查**，结果仅作健康参考，不能替代临床诊断。

### 2. 测验功能
可区分存在心理困扰、心理障碍边缘的人群，辅助排查身心相关问题；既可自测，也可辅助他人心理状态核查，高分提示需寻求专业心理/精神科帮助。

## 三、统计计算指标
1. **总分**：90个条目得分全部相加总和
2. **总症状指数（总均分）**：总分 ÷ 90
3. **阳性项目数**：评分≥2分（轻度及以上）的条目数量；阳性痛苦水平 = 总分 ÷ 阳性项目数
4. **阳性症状均分**：(总分 − 所有评1分阴性条目总分) ÷ 阳性项目数
