## 二、计分规则（后台测评使用，不对测评用户完整展示）
### 1. 总分分级判定
|总分区间|筛查结果|参考建议|
| ---- | ---- | ---- |
|0～4分|没有抑郁相关倾向|日常留意身心状态，规律作息|
|5～9分|可能存在轻度抑郁倾向|建议咨询心理相关从业者|
|10～14分|可能存在中度抑郁倾向|最好面诊心理医师进行评估|
|15～19分|可能存在中重度抑郁倾向|建议前往医院心理/精神科就诊|
|20～27分|可能存在重度抑郁倾向|务必尽快到精神专科就医|

### 2. 核心项目专项判定
1. 核心题：**第1题、第4题（抑郁核心症状）、第9题（轻生意念）**
2. 判定标准：上述任意一题得分＞1分（选择2分、3分），需要重点关注测评人状态。

> 小程序备注：本量表仅为心理健康筛查参考，不能作为临床诊断，症状持续请前往正规医院就诊。
