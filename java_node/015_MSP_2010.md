# 参考答案
## 一、填空题（每题1分，共20分）
1. 在信号处理领域，应用最多的三种神经网络模型是：**BP神经网络、RBF神经网络、自组织特征映射神经网络（SOFM）**
2. 自组织特征映射（SOFM）包括三个基本过程是：**竞争、合作、更新**
3. 如果平稳随机过程是各态历经的，可以用**时间平均**代替**统计平均**。
4. 维纳-辛钦定理表明，随机信号的自相关函数的傅里叶变换是**功率谱密度**。
5. 白噪声过程的方差为$\sigma^{2}$，则其功率谱$S(e^{j \omega})=$**$\sigma^2$**，自相关函数$R(m)=$**$\sigma^2\delta(m)$**。
6. 在三种有理分式模型中，**MA**模型是全零点模型，AR模型应用较为广泛，这主要是因为：(1)**可以通过线性方程求解**；(2)**谱估计分辨率高**。
7. LMS算法中，在满足收敛条件情况下，选择步长因子要兼顾**收敛速度**和**稳态误差**。
8. 设滤波器的系统函数$H(z)=1-2 z^{-1}+3 z^{-2}+4 z^{-3}$，利用多相分解将$H(z)$表示为2个子滤波器组成的二相网络结构，则这两个子滤波器的系统函数为$E_{0}(z)=$**$1+3z^{-1}$**，$E_{1}(z)=$**$-2+4z^{-1}$**。
9. “先插值，后滤波”中的滤波是为了**去除镜像频谱**，“先滤波，后抽取”中的滤波是为了**防止混叠**。

## 三、简答题（每题5分，共20分）
### 1. 基于AR模型的谱估计步骤
1. 确定AR模型阶数$p$；
2. 由观测数据估计自相关函数；
3. 解Yule‑Walker方程得到AR系数；
4. 代入功率谱公式$P_{AR}(e^{j\omega})=\frac{\sigma^2}{|1+\sum_{k=1}^p a_k e^{-jk\omega}|^2}$得到谱估计。

### 2. 高阶谱抗有色正态噪声的原因
高斯噪声的三阶及以上累积量恒为0，而非高斯信号高阶累积量非零；高阶谱只保留非高斯信息，可抑制高斯有色噪声。

### 3. 连续小波变换的变焦距性质
小波变换在**高频段时域分辨率高、频域分辨率低**，适合紧盯瞬态；**低频段频域分辨率高、时域分辨率低**，适合观察整体趋势，类似变焦。

### 4. LMS与最陡下降法的主要不同
最陡下降法用**精确梯度**；LMS用**单次采样梯度估计**，计算简单但有梯度噪声，收敛稍慢。

## 四、画图说明题（共14分）
### 1. 抽取与内插频谱
1. **M=2直接抽取频谱**：原频谱$[-\frac{2\pi}{3},\frac{2\pi}{3}]$，抽取后周期延拓，在$\pm\frac{\pi}{2}$附近出现**混叠**。
2. **L=2直接插零频谱**：原频谱压缩为$[-\frac{\pi}{3},\frac{\pi}{3}]$，并在$\pm\pi$附近出现**镜像频谱**。

### 2. 自适应系统辨识结构
输入$x(n)$ → 待辨识系统 + 自适应滤波器 → 输出相减得到误差$e(n)$ → 误差控制滤波器权值更新。

## 五、综合题（每题12分，共36分）
### 1. 一阶线性预测器求解
已知$x(n)=\{1,2,3,4,5\}$，$h(0)=1$，预测误差$e(n)=x(n)+h(1)x(n-1)$。
自相关：$r(0)=\frac{1}{5}\sum x^2(n)=11$，$r(1)=\frac{1}{5}\sum x(n)x(n-1)=7$。
Yule‑Walker：$r(1)+h(1)r(0)=0$ → $h(1)=-\frac{7}{11}$。

### 2. AR(2)参数与极点
反射系数$k_1=-\frac{5}{7}$，$k_2=\frac{3}{25}$。
由格型转AR递推：
$a_1=k_1=-\frac{5}{7}$
$a_2=k_2=-\frac{3}{25}$
系统函数$H(z)=\frac{1}{1-\frac{5}{7}z^{-1}-\frac{3}{25}z^{-2}}$
极点：$z^2-\frac{5}{7}z-\frac{3}{25}=0$，解得$z_1,z_2$。

### 3. 双输入维纳滤波
1. **用途**：从两路不相关噪声观测中**最优恢复原始信号$s(n)$**。
2. **优势**：两路噪声不相关，更满足维纳滤波**噪声互不相关**的假设，估计误差更小。
